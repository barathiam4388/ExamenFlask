# Examen - Application Flask (Bibliothèque)

Projet Flask avec templates HTML et base SQLite.

## Technologies
- Python
- Flask
- SQLite

## Structure
- `app.py` : application Flask
- `classe.py` : logique / classes
- `templates/` : pages HTML
- `library_db.sqlite` : base de données (si incluse)

## Lancer le projet (recommandé)
1. Créer un environnement virtuel
2. Installer Flask
3. Lancer `python app.py` ou `flask run`
